Retro Terminals

Thirty-nine live iTerm2 profiles in four packs — real machines, fictional computers, aesthetic movements, and the Alien/Blade Runner megacorps. Every card is rendered in that profile's real installed font and exact colors. Pick one from ⌘O and filter by tag. Fonts marked * use Departure Mono — originals aren't freely hosted.

How the kit fits together — one generator (build_profiles.py), five moving parts:
  1. Profiles — palette + font + cursor, hot-loaded live by iTerm2
  2. Boot banners — each profile prints its startup screen (Initial Text)
  3. retro <machine> — flips your live shell prompt to match (starship-aware)
  4. Scanline bezel — a CRT overlay drawn behind the text on the tube profiles
  5. CRT playground — the browser-only part →
▶ CRT Playgroundbend text through curved glass

The playground does the one thing iTerm can't: a real WebGL curved-tube shader (barrel distortion, phosphor bloom, aperture mask). The bezel overlay above is iTerm's closest native approximation — texture behind text, not text bent through glass.

Use one

Profiles menu, or ⌘O, filter by retro / sci-fi / aesthetic. Set a default in Settings ▸ Profiles ▸ Other Actions.

Match the prompt

source retro-prompts.zsh then retro c64, retro hal, retro steampunk… retro off restores starship.

Tweak

Edit hex in build_profiles.py, re-run, tabs restyle within a second — iTerm2 hot-loads the JSON.
